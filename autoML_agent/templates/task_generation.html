    ML 태스크 선택

  • 특정 데이터가 A이거나 B인가요? 에서는 분류 알고리즘을 사용합니다.
  • 데이터의 양 또는 수치를 예측하나요? 에서는 회귀 알고리즘을 사용합니다.
  • 데이터가 어떻게 구성되어 있나요? 에서는 클러스트링(군집화) 알고리즘을 사용합니다.
  • 데이터가 이상한가요? 에서는 이상치 탐지 알고리즘을 사용합니다.
  • 어떻게 해야 하나요? 에서는 강화학습 알고리즘을 사용합니다.

    데이터가 준비되어 있나요?

  • 데이터가 준비되어 있다면 기존의 준비되어 있는 데이터를 사용합니다.
  • 만약, 데이터가 준비되어 있지 않다면 새로운 데이터를 불러옵니다.
